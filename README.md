# SkyForge — Forge Your Own Destiny

The AI-powered game world builder. Describe any world in plain language — SkyForge generates a fully realized game universe complete with regions, lore, characters, factions, and secrets.

## Stack
- Next.js 15 + React 19
- TypeScript + Tailwind CSS
- Gemini 1.5 Flash (world generation + NPC dialogue)

## Features
- **World Forge** — text prompt → complete game world (regions, towns, routes, landmarks, characters)
- **NPC Dialogue** — live AI-powered character conversations
- **GOD MODE** — all genre restrictions removed. Any world. Any destiny.
- **Shareable Worlds** — every forge result is unique and shareable

## Getting Started
```bash
npm install
npm run dev
```

Open http://localhost:4028

## Environment Variables
```
GEMINI_API_KEY=your_key_here
NEXT_PUBLIC_SITE_URL=https://yourdomain.gg
```
